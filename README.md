# Deep Learning for Medical Image Analysis – CNN Classification & U-Net Segmentation

This repository contains two independent deep learning projects focused on medical image analysis.

🩸 Project 1 – HER2 Classification:
Breast Cancer Immunohistochemical (BCI) dataset classification using a custom CNN architecture.
The model predicts HER2 scores (0, 1+, 2+, 3+) with class-balanced training, data augmentation, and TensorBoard-based performance tracking.

🔬 Project 2 – TNBC Nuclei Segmentation:
Nuclei segmentation on TNBC MS Bench dataset using a U-Net encoder–decoder model.
The network combines BCE + Dice + Focal Loss functions and achieves strong pixel-level accuracy and IoU scores.

Both projects demonstrate practical applications of convolutional neural networks in medical image classification and segmentation, emphasizing balanced training, visual evaluation, and model generalization.
